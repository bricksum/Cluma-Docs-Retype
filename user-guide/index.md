## 기본 사용 흐름
 
Cluma를 처음 사용하는 경우 아래 순서로 진행하는 것을 권장합니다.
 
**1. 스토리지 생성**
 
학습 데이터나 모델 파일을 영구적으로 저장할 스토리지를 먼저 생성합니다. 컨테이너를 삭제하더라도 스토리지에 저장된 데이터는 유지됩니다.
 
**2. SSH Key 등록**
 
프로필 → SSH Key 관리에서 공개키를 등록합니다. 컨테이너 생성 시 Key를 선택하면 비밀번호 없이 SSH 접속이 가능합니다.
 
**3. 컨테이너 생성**
 
이미지, 리소스(CPU / 메모리 / GPU), 포트 포워딩, 스토리지 마운트를 설정하고 컨테이너를 생성합니다.
 
**4. 개발 환경 접속**
 
생성된 컨테이너에 SSH, JupyterLab, VSCode 등으로 접속하여 작업을 시작합니다.
 
---
 
## 리소스 사용 현황 확인
 
대시보드 상단 헤더에서 현재 그룹의 CPU, 메모리, GPU, 스토리지 잔여량을 실시간으로 확인할 수 있습니다. 잔여 리소스가 부족한 경우 관리자에게 한도 증가를 요청합니다.
