# 설정

프로필 정보, 비밀번호, SSH Key를 관리합니다.

---

## 프로필 정보

이름과 이메일을 확인하고 수정할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 이름 | 플랫폼 내 표시 이름 |
| 이메일 | 계정 이메일 주소 |
| 소속 그룹 | 현재 소속된 그룹 (변경 불가, 관리자 권한 필요) |

---

## 비밀번호 변경

새 비밀번호를 입력하고 확인란에 동일하게 입력한 뒤 **저장** 버튼을 클릭합니다.

---

## SSH Key 관리

SSH Key를 등록하면 컨테이너에 SSH로 안전하게 접속할 수 있습니다.

### SSH Key 등록

사이드바 하단 **설정** → **SSH Key 추가** 버튼을 클릭합니다.

| 항목 | 설명 |
|------|------|
| SSH Key 이름 | Key를 식별하는 이름 (예: My Laptop Key) |
| SSH 공개키 | 직접 입력 또는 파일 업로드 |

- **직접 입력** : 공개키 텍스트를 붙여넣습니다. (`ssh-rsa AAAA...` 형식)

- **파일 업로드** : `.pub` 파일을 선택하여 업로드합니다.

### SSH Key 삭제

등록된 Key 목록에서 삭제 아이콘을 클릭합니다.

> 삭제된 SSH Key는 해당 Key로 접속 중인 컨테이너에 즉시 반영됩니다.
