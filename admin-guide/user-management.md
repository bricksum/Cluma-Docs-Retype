# 사용자 관리

사용자 생성·수정·삭제 및 역할 설정 방법을 설명합니다.

---

## 사용자 목록

관리자 메뉴 → **사용자 관리**에서 전체 사용자 목록을 확인할 수 있습니다.

| 컬럼 | 설명 |
|------|------|
| 이메일 | 로그인에 사용하는 이메일 주소 |
| 이름 | 사용자 표시 이름 |
| 역할 | 관리자 또는 사용자 |
| 그룹 | 소속된 그룹 목록 |
| 작업 | 수정(연필 아이콘) / 삭제(휴지통 아이콘) |

---

## 사용자 생성

우측 상단 **사용자 추가하기** 버튼을 클릭합니다.

| 항목 | 설명 | 필수 여부 |
|------|------|-----------|
| 이름 | 사용자 표시 이름 | 필수 |
| 이메일 | 로그인에 사용할 이메일 주소 | 필수 |
| 비밀번호 | 초기 비밀번호 설정 | 필수 |
| 역할 | 관리자 또는 사용자 | 필수 |
| 그룹 | 소속 그룹 (여러 개 선택 가능) | 선택 |

생성 시 그룹을 지정하지 않으면 **Default 그룹**에 자동으로 배정됩니다.

---

## 사용자 수정

목록에서 수정할 사용자의 **연필 아이콘**을 클릭합니다.

이름, 이메일, 역할을 수정할 수 있습니다. **비밀번호 초기화** 버튼을 통해 해당 사용자의 비밀번호를 초기화할 수 있습니다.

---

## 사용자 활성화 / 비활성화
 
계정을 일시적으로 사용 중지할 수 있습니다.

- **비활성화**: 해당 사용자는 로그인이 불가능해지며, 실행 중인 컨테이너는 자동으로 정지됩니다.
- **활성화**: 비활성화된 계정을 다시 사용 가능한 상태로 전환합니다.

!!!warning 사용자 비활성화 시 주의
비활성화 시 해당 사용자의 실행 중인 컨테이너가 정지됩니다. 중요한 작업이 진행 중인지 사전에 확인하세요.
!!!

---

## 사용자 삭제

목록에서 삭제할 사용자의 **휴지통 아이콘**을 클릭합니다. 삭제 확인을 위해 **사용자 이름을 직접 입력**해야 합니다.

!!!danger 삭제 시 주의
이 작업은 되돌릴 수 없습니다. 사용자를 삭제하면 해당 사용자의 컨테이너와 개인 스토리지가 함께 삭제될 수 있습니다.
!!!

---
