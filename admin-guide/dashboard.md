# 관리자 대시보드

클러스터 전체 리소스 사용 현황과 그룹별 사용량을 통합 모니터링하는 화면입니다.

관리자 메뉴 → **관리자 대시보드**에서 접근합니다.

---

## 클러스터 전체 현황

상단 **클러스터 전체 현황** 섹션에서 리소스 상태를 카드로 확인합니다.

각 카드는 **사용량 / 할당량 / 총량** 및 **사용률** 프로그레스 바로 표시됩니다.

| 카드 | 단위 |
|------|------|
| CPU | cores |
| Memory | GB |
| Storage | GB |
| 디바이스별 현황 | 개 (가속기 유형별로 개별 카드 표시) |

사용률에 따라 프로그레스 바와 수치 색상이 변경됩니다.

| 범위 | 색상 |
|------|------|
| 0 ~ 69% | 기본 |
| 70 ~ 89% | 노란색 |
| 90% 이상 | 빨간색 |

---

## 그룹별 리소스 사용량

하단 **그룹별 리소스 사용량** 섹션에서 그룹 단위 사용 현황을 확인합니다.

### 그룹 필터

상단 드롭다운에서 그룹을 선택하면 파이차트 표시 내용이 전환됩니다.

| 선택 | 파이차트 표시 내용 |
|------|------|
| 전체 | 클러스터 전체 용량 대비 각 그룹의 할당량 비율 |
| 특정 그룹 | 해당 그룹의 할당량 대비 현재 사용량 |

파이차트에서 그룹 영역을 직접 클릭해도 해당 그룹 상세 보기로 전환되며, **← 전체 보기** 버튼으로 돌아올 수 있습니다.

파이차트 위에 마우스를 올리면 해당 그룹의 할당량과 비율이 툴팁으로 표시됩니다.

### 파이차트

CPU / Memory / 가속기 / Storage 4개 리소스에 대해 그룹별 비율을 파이차트로 시각화합니다.  
미할당 용량은 회색으로 표시됩니다.

### 그룹별 요약 테이블

모든 그룹의 리소스 사용 현황을 한눈에 확인할 수 있습니다.

| 열 | 설명 |
|------|------|
| 그룹명 | 그룹 이름 (클릭 시 해당 그룹 상세 보기로 전환) |
| CPU 사용/할당 | CPU 코어 사용량 / 할당량 및 사용률(%) |
| MEMORY 사용/할당 | 메모리(GB) 사용량 / 할당량 및 사용률(%) |
| 디바이스 사용/할당 | 가속기 유형별 사용량 / 할당량 |
| STORAGE 사용/할당 | 스토리지(GB) 사용량 / 할당량 및 사용률(%) |
| POD 수 | 현재 실행 중인 Pod 수 |

사용률이 70% 이상이면 노란색, 90% 이상이면 빨간색으로 강조됩니다. 할당량이 없는 경우 `(미할당)` 으로 표시됩니다.
