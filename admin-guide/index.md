# 관리자 가이드

관리자 계정에서 사용할 수 있는 운영 기능을 설명합니다.

이 섹션의 기능은 관리자(Admin) 계정에서만 접근할 수 있습니다. 일반 사용자 계정에서는 해당 메뉴가 표시되지 않습니다.

---
 
## 관리자란?
 
관리자는 Cluma 플랫폼의 전체 운영 권한을 가진 사용자입니다. 일반 사용자와 달리 그룹에 소속되지 않아도 되며, 플랫폼의 모든 리소스와 설정에 접근할 수 있습니다.
 
---

## 관리자 기능 목록

| 기능 | 설명 |
|------|------|
| 유저 / 그룹 관리 | 유저 생성·수정·삭제, 그룹 생성·리소스 한도 설정, 유저-그룹 할당 |
| 노드 관리 | 클러스터 노드 등록·수정·활성화, GPU 모델·MIG 설정 |
| 레지스트리 관리 | 컨테이너 이미지 레지스트리 등록 및 관리 |
| 관리자 대시보드 | 전체 유저·그룹의 리소스 사용량 통합 모니터링 |
| 감사 로그 | 모든 유저·관리자 액션 이력 조회 |
