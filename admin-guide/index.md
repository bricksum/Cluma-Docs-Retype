# 관리자 가이드

관리자 계정에서 사용할 수 있는 운영 기능을 설명합니다.

이 섹션의 기능은 관리자(Admin) 계정에서만 접근할 수 있습니다. 일반 사용자 계정에서는 해당 메뉴가 표시되지 않습니다.

---
 
## 관리자란?
 
관리자는 Cluma 플랫폼의 전체 운영 권한을 가진 사용자입니다. 일반 사용자와 달리 그룹에 소속되지 않아도 되며, 플랫폼의 모든 리소스와 설정에 접근할 수 있습니다.
 
---

## 관리자 기능 목록

| 기능 | 설명 |
|------|------|
| 클러스터 대시보드 | 전체 그룹의 리소스 사용량 통합 모니터링 |
| 사용자 관리 | 사용자 생성·수정·삭제, 역할 설정 |
| 그룹 관리 | 그룹 생성·리소스 한도 설정, 사용자-그룹 할당 |
| 레포지토리 관리 | 컨테이너 이미지 레포지토리 등록 및 관리 |
| 노드 현황 | 클러스터 노드 상태 및 Pod 현황 실시간 확인 |
| 클러스터 모니터링 | Grafana 기반 클러스터 실시간 모니터링 |
