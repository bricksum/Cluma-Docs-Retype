# Getting Started

Cluma를 처음 사용하는 분을 위한 시작 가이드입니다.

---

## 역할별 시작 가이드

### 일반 사용자

그룹에 배정된 리소스 범위 안에서 컨테이너를 생성하고 GPU 개발 환경을 사용합니다.

→ [사용자 빠른 시작](./quickstart.md)

### 관리자

사용자 계정을 만들고, 그룹에 리소스 한도를 설정하고, 레포지토리를 등록합니다.

→ [관리자 초기 설정](./admin-setup.md)

---

## 사전 요구사항

Cluma를 클러스터에 배포하려면 Kubernetes 환경과 필수 도구가 필요합니다.

→ [사전 요구사항](./prerequisites.md)
